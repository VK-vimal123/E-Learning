# E-Learning Website (Frontend Only)

Simple E-Learning app using only HTML, CSS, and JavaScript.

## How to run
1. Clone or download the project folder.
2. Open `index.html` in your browser (double-click or right-click > Open With > Browser).

No backend or build steps required.

---

Files:
- index.html — Home page
- courses.html — Course list
- lesson.html — Lesson detail with embedded video and notes
- quiz.html — Quiz page with MCQs and score calculation
- assets/css/style.css — Styles
- assets/js/data.js — Course and quiz data
- assets/js/app.js — App logic
